Hotlinks

Responsável pela configuração dos hotlinks do app.




Objetivo

Autonomia operacional para alterar os hotlinks dos apps.



Tela


Marca
É necessário definir os hotlinks para as duas marcas (Zoom/Buscapé).
Cada marca fica separada na aba superior da tela.



Título
Defina o texto de será exibido embaixo do icone do hotlink.
Existem 4 tipos de hotlinks:

1 - Categoria
Ao digitar no campo irá exibir uma lista de categoria, então basta clicar na categoria escolhida.
Caso não apareca a categoria que deseja, verificar se existe no banco de dados do Zoom.
O hotlink levará o usuário para a tela de CatProd ou CatOffer da categoria selecionada.

2 - Busca
Existem dois campos.
- Termo da busca: Query que será enviada para a tela da Algolia para exibir os resultados.
- Título: Texto que aparecerá embaixo do ícone do hotlink.
O hotlink levará o usuário para a tela de resultado de busca da query preenchida no campo.



3 - Pechinchas

Define o texto que aparecerá embaixo do ícone do hotlink.
O hotlink levará o usuário para a tela de pechinchas.

4 - Ver todos
Define o texto que aparecerá embaixo do ícone do hotlink.
O hotlink levará o usuário para a tela de categorias do app.





Preview
Será exibido um preview de como serão exibidos os hotlinks nos apps.

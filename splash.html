Splash

Responsável pela imagem da tela de Splash do app.




Objetivo

Autonomia operacional para alterar a primeira imagem que é exibida no app(Splash) de acordo com agendamento realizado na intranet.



Tela


Nome do Projeto
Defina um nome para o projeto que será exibido na listagem e a marca (Zoom/Buscapé)



Imagens
Defina a imagem que será exibido no app. Deverá ser nos formatos png,jpg ou jpeg.
São 3 tamamhos necessários: 412x732 (1x) | 824x1464 (2x) | 1236x2196 (3x)



Agendamento
Defina a data de início e data final, na qual a imagem estará sendo exibida no app.
A data final não é obrigatória, logo a imagem ficará sempre disponível (caso o projeto esteja ativo) e não exista um projeto no mesmo período selecionado.



Listagem
Será exibido todos os projetos criados. Na coluna status (última coluna) é exibido se o projeto está ativo(verde) ou desativado.
